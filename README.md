# Training Management System

This is a full-stack Training Management System web application.

## Features
- Role-based access: participant, staff, admin
- Program management (CRUD)
- Participant registration and payment verification
- Attendance marking and certificate generation
- Real-time updates with Socket.IO
- Admin dashboard statistics

## Run locally
1. Install Node.js (v18+)
2. Run `npm install`
3. Copy `.env.example` to `.env` and fill in your MongoDB connection
4. Run `npm run dev`
5. Access http://localhost:4000
